## Requirements

- Python 3.6 or higher.

## Run

1. `git clone https://github.com/manangadwal/Python-Assistant-Active`, to download the source code.
2. `cd Python-Assistant-Active`, to enter the directory.
3. Run with `python3 active.py`.
